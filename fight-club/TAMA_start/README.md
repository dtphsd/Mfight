# 🧬 Evolution Tamagotchi — Starter Kit

> **Что это?** Система геймификации роста AI-агента. Ты кодишь, агент учится на ошибках, а тамагочи растёт 🥚→🦊→🐉→⭐

---

## 🚀 Быстрый старт (5 минут)

### Шаг 1: Скопируй файлы в свой проект

```
Скопируй из этой папки в КОРЕНЬ своего проекта:

📁 твой-проект/
├── .agents/
│   └── workflows/
│       └── auto-evolution.md      ← правила для AI-агента
├── evolution_journal.md           ← журнал эволюции (сюда пишутся записи)
└── ... (твои файлы проекта)
```

### Шаг 2: Кодь как обычно

Ничего не меняется! Продолжай работать в **VS Code**, **Antigravity** или любом другом редакторе с AI-агентом.

AI-агент будет автоматически:
- 📝 Добавлять EVO-записи в `evolution_journal.md` когда фиксит баги
- 🧠 Извлекать паттерны (что ДЕЛАТЬ) и анти-паттерны (что НЕ делать)
- 📊 Начислять XP и повышать уровень

### Шаг 3: Смотри прогресс

1. Открой `visualizer/index.html` в браузере
2. Перетащи свой `evolution_journal.md` на страницу (или нажми **⚡ Demo**)
3. Наслаждайся тамагочи! 🎮

## ❓ FAQ

**Q: Нужен ли сервер?**
A: Нет. Всё локально — 1 HTML файл, 0 зависимостей.

**Q: Какой AI-агент нужен?**
A: Любой! Antigravity, GitHub Copilot, Cursor, Cline... Главное — чтобы он читал `.agents/workflows/auto-evolution.md`.

**Q: Что если агент не пишет в журнал?**
A: Попроси его: *"Обнови evolution_journal.md по RULE 8 из auto-evolution.md"*

**Q: Как поделиться прогрессом с друзьями?**
A: Фаза 2 (в разработке) — GitHub Gist API + Leaderboard. Пока можно просто показать скриншот!
